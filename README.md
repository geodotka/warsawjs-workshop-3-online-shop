# boilerplate-babel-webpack

> :fork_and_knife: Starter with Babel.js and Webpack

## Features

* [x] Webpack v2.2.1
* [x] Babel Core v6.22.1
* [x] Static directory: `public/`

## How to run?

```
$ npm install
$ npm run build
# Open public/ directory in browser
```
